# Under World

## ゲームの内容
敵の攻撃を避けながら、それを利用して敵の倒す弾幕風避けゲーです。

![画面イメージ](docs/images/game_image01.png)

## 操作説明
- WASDで上下左右に移動する
- メニューは、自機をボタンの上に合わせスペースキーで選択出来ます。
